:py:mod:`evutils.vis.plot3d`
============================

.. py:module:: evutils.vis.plot3d

.. autoapi-nested-parse::

   Matplotlib 3D visualization utilities for event data.

   This module provides functions to plot event streams, 3D histograms,
   and time surfaces using matplotlib's 3D plotting capabilities.

.. py:function:: plot_3d(events: numpy.ndarray, width: int = 1280, height: int = 720, colormap: Union[str, matplotlib.colors.Colormap] = 'Spectral', fig: Optional[matplotlib.pyplot.Figure] = None, ax: Optional[mpl_toolkits.mplot3d.Axes3D] = None) -> tuple

   Plot a 3D scatter plot of events.

   :param events: Array of events with fields 't', 'x', 'y', and 'p'.
   :type events: np.ndarray
   :param width: Width of the event frame, by default 1280.
   :type width: int, optional
   :param height: Height of the event frame, by default 720.
   :type height: int, optional
   :param colormap: Colormap to use for the event polarities, by default 'Spectral'.
   :type colormap: Union[str, Colormap], optional
   :param fig: Matplotlib figure to plot on, by default None (a new figure will be created).
   :type fig: Optional[plt.Figure], optional
   :param ax: Matplotlib 3D axis to plot on, by default None (a new axis will be created).
   :type ax: Optional[Axes3D], optional

   :returns: A tuple containing the figure and axis objects.
   :rtype: tuple

   :raises ValueError: If the colormap is not found or invalid.

   Plot a 3D time surface of events.
