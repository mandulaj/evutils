.. py:function:: histogram(events: numpy.ndarray, width: int = 1280, height: int = 720, fill: bool = False, dtype: Any = np.uint8) -> numpy.ndarray

   Generate a histogram frame from the events.

   :param events: Array of events in the :class:`~evutils.types.Events` format
   :type events: np.ndarray
   :param width: Width of the frame, by default 1280
   :type width: int, optional
   :param height: Height of the frame, by default 720
   :type height: int, optional
   :param fill: If True, the non-zero values are set to 255, by default False
   :type fill: bool, optional
   :param dtype: Data type of the output array, by default np.uint8
   :type dtype: np.dtype, optional

   :returns: A numpy array with the histogram frame (height, width, 3)
   :rtype: np.ndarray

   .. rubric:: Examples

   >>> import numpy as np
   >>> from evutils.repr import histogram
   >>> events = np.array([(10, 20, 100, 1), (15, 25, 200, 0)],
   ...                   dtype=[('x', '<u2'), ('y', '<u2'), ('t', '<i8'), ('p', 'i1')])
   >>> frame = histogram(events, width=100, height=100)
   >>> frame.shape
   (100, 100, 3)

.. py:function:: voxel_histogram(events: numpy.ndarray, width: int = 1280, height: int = 720, n_bins: int = 10, dt: int = 10000, dtype: Any = np.uint8) -> numpy.ndarray

   Generate a voxel grid from the events.

   :param events: Array of events in the :class:`~evutils.types.Events` format.
   :type events: np.ndarray
   :param width: Width of the voxel grid, by default 1280.
   :type width: int, optional
   :param height: Height of the voxel grid, by default 720.
   :type height: int, optional
   :param n_bins: Number of depth bins (time slices) in the voxel grid, by default 10.
   :type n_bins: int, optional
   :param dt: Time delta in microseconds for events buffer by default 10_000 (10 ms).
   :type dt: int, optional
   :param dtype: Data type of the output voxel grid, by default np.uint8.
   :type dtype: np.dtype, optional

   :returns: A numpy array with the voxel grid (n_bins, height, width, 3).
   :rtype: np.ndarray

   .. rubric:: Examples

   >>> import numpy as np
   >>> from evutils.repr import voxel_histogram
   >>> events = np.array([(10, 20, 100, 1), (15, 25, 200, 0), (20, 30, 10000, 1)],
   ...                   dtype=[('x', '<u2'), ('y', '<u2'), ('t', '<i8'), ('p', 'i1')])
   >>> grid = voxel_histogram(events, width=100, height=100, n_bins=10, dt=10000)
   >>> grid.shape
   (10, 100, 100, 3)

.. py:function:: tore(events: numpy.ndarray, width: int = 1280, height: int = 720, n_events: int = 4, tau: int = 10000, dtype: Any = np.uint8) -> numpy.ndarray

   Generate a TORE from the events.

   :param events: Array of events in the :class:`~evutils.types.Events` format
   :type events: np.ndarray
   :param width: Width of the frame, by default 1280
   :type width: int, optional
   :param height: Height of the frame, by default 720
   :type height: int, optional
   :param n_events: Number of events to keep in the TORE, by default 4
   :type n_events: int, optional
   :param tau: Time constant for the exponential decay, by default 10_000
   :type tau: int, optional
   :param dtype: Data type of the output array, by default np.uint8
   :type dtype: np.dtype, optional

   :returns: A numpy array with the TORE representation (height, width, n_events, 2)
   :rtype: np.ndarray

   .. rubric:: Examples

   >>> import numpy as np
   >>> from evutils.repr import tore
   >>> events = np.array([(10, 20, 100, 1), (15, 25, 200, 0)],
   ...                   dtype=[('x', '<u2'), ('y', '<u2'), ('t', '<i8'), ('p', 'i1')])
   >>> frame = tore(events, width=100, height=100, n_events=4)
   >>> frame.shape
   (100, 100, 4, 2)

   [1] Baldwin, R. W., Liu, R., Almatrafi, M., Asari, V., & Hirakawa, K. (2022). Time-ordered recent event (tore) volumes for event cameras. IEEE Transactions on Pattern Analysis and Machine Intelligence, 45(2), 2519-2532.
