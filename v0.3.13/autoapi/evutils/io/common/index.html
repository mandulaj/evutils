evutils.io.common#

Common interface for event decoders and encoders.

Defines the abstract base classes EventDecoder and EventEncoder.

Classes#

ABC for reading chunks of events from a IO source object.

read_all() → EventArray | tuple[EventArray, TriggerArray][source]#

Decode and return every remaining event at once.

The default implementation drains read_chunk() and concatenates the chunks. SoA-native decoders (EVT/DAT/AER) override this with a single-buffer decode that avoids the per-chunk copy entirely.

Returns:

All remaining events.

Return type:

EventArray
